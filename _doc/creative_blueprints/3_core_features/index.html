What it does

Core Features

Parent Child
⏰

Alarm Setup

Configure wake-up time and a kid-friendly sound. School-day schedule awareness means it fires only on school mornings — skip weekends or custom days off automatically.

📋

Routine Builder

Add, name, and order every morning step — get dressed, brush teeth, eat breakfast, pack bag. Set duration hints so the child knows how long each task should take.

⭐

Rewards & Celebration

A star trail progress bar fills as each step is ticked off. Finish all steps and a confetti celebration fires — positive reinforcement only, no failure states, no scolding.

📊

Parent Dashboard

PIN-protected parent view shows today's step-by-step breakdown and a 7-day colour-coded history grid — green for all done, yellow for partial. No nagging needed.
